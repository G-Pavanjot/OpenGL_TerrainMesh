Commands:
q - quit
w - wireframe toggle
r - reset
+ - increase size
- - decrease size
s - shademodel change
l - light toggle
0 - select light0 to move
1 - select light1 to move
t - increase x pos of selected light
y - decrease x pos of selected light
u - increase y pos of selected light
i - decrease y pos of selected light
o - increase z pos of selected light
p - decrease z pos of selected light

left - rotate about y axis
right - rotate about y axis
down - rotate about x axis
up - rotate about x axis

c - custom terrain mode


when in custom terrain mode, user clicks on the 2D overview to create the points used to create a custom line for the fault algorithm
